WAS MACHT DIESES MODUL?

Dieses Modul ermöglicht die Anzeige der in der Shop-Administration definierten Sprachsymbole (Flaggen) und Währungssymbole als Links im Navigationsbereich des Headers.

Voraussetzung ist natürlich, dass mindestens eine zusätzliche Sprache oder Währung eingerichtet wurde.

Die Anzeige im Header kann unter
Admin → Konfiguration → Layout-Einstellungen
ein- oder ausgeschaltet werden.

========================================================

Bitte erstellen Sie vor der Installation unbedingt eine Sicherung Ihrer Dateien sowie der SQL-Datenbank und führen Sie anschließend die Dateien dieses Moduls mit Ihrer bestehenden Zen-Cart-Installation zusammen.

INSTALLATION

Schritt 1:

Laden Sie alle Dateien aus dem Verzeichnis „NEUE-DATEIEN“ auf den Server hoch.

Schritt 2:

Laden Sie alle Dateien aus dem Verzeichnis „GEAENDERTE-DATEIEN“ auf den Server hoch.

========================================================

DEINSTALLATION

Schritt 1:

Entfernen Sie die mit diesem Modul installierten Dateien von Ihrem Server. Stellen Sie dabei sicher, dass zusammengeführte Dateien wie

includes/templates/TEMPLATE_NAME/common/tpl_header.php

durch die zuvor gesicherten Originaldateien ersetzt werden.

Schritt 2:

Führen Sie die folgenden SQL-Anweisungen über

Admin → Tools → SQL-Patches installieren

aus, um die Konfigurationseinträge aus der Datenbank zu entfernen:

DELETE FROM configuration WHERE configuration_key = 'HEADER_LANGUAGES_DISPLAY';
DELETE FROM configuration WHERE configuration_key = 'HEADER_CURRENCIES_DISPLAY';
