Version 5.0 für deutsche Zen Cart Pro 157k (zen-cart-pro.at)

========================================================

WAS MACHT DIESES MODUL?

Dieses Modul ermöglicht die Anzeige der im Shop-Admin definierten Sprachsymbole (Flaggen) und Währungssymbole als Links im Navigationsbereich des Headers.

Voraussetzung ist eine konfigurierte zusätzliche Sprache oder Währung.

Die Anzeige im Header kann unter
Admin → Konfiguration → Layout-Einstellungen
ein- oder ausgeschaltet werden.

========================================================

INSTALLATION

Vor der Installation eine Sicherung der Dateien und der Datenbank erstellen. Anschließend die Dateien dieses Moduls mit der bestehenden Zen-Cart-Installation zusammenführen.

Schritt 1:
Verzeichnis DEIN-ADMIN/ entsprechend der tatsächlichen Shop-Admin-Struktur umbenennen.

Schritt 2:
Dateien mit der bestehenden Installation auf dem Server abgleichen.
In vielen Installationen betrifft dies primär:

- tpl_header.php

Hinweis:
Zum Vergleich eine Datei z. B. von tpl_header.php in tpl_header.php.bak umbenennen und lokal herunterladen.
Anschließend beide Versionen mit einem Diff-Tool (z. B. Meld) vergleichen und Änderungen gezielt übernehmen.

Schritt 3:
Nach dem Merge alle Dateien auf den Server hochladen.
Bei Nutzung von responsive_classic sind keine Anpassungen an anderen Templates erforderlich.

========================================================

DEINSTALLATION

Schritt 1:
Installierte Modul-Dateien vom Server entfernen.
Zusammengeführte Dateien wie:

includes/templates/TEMPLATE_NAME/common/tpl_header.php

durch die gesicherten Originaldateien ersetzen.

Schritt 2:
SQL-Anweisungen über
Admin → Tools → SQL-Patches installieren
ausführen:

DELETE FROM configuration WHERE configuration_key = 'HEADER_LANGUAGES_DISPLAY';
DELETE FROM configuration WHERE configuration_key = 'HEADER_CURRENCIES_DISPLAY';
